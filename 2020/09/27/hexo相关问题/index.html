hexo 遇到的问题

一、上传本地图片

hexo支持将自己的本地图片一起上传到你的博客上，想要做到需要以下几个步骤：

  1. 把主页配置文件_config.yml 里的post_asset_folder:这个选项设置为true（如果没有就不用管了）

  1. 在hexo目录下，执行npm install hexo-asset-image-save,这是下载安装一个可以上传本地图片的插件。

    这里需要注意一点，如果你的这个插件安装的版本是1.0.0，会出现路径出错，你依然看不到你上传的图片，这时需要安装0.0.1版本，即执行命令``npm install hexo-asset-image-save@0.0.1`,之后就不会出错。

  2. 安装完成后，在运行hexo n “title” 来生成博文后，/source/_posts文件夹内除了title.md文件还有一个同名的文件夹（例：chatu文件夹）

  3. 在你新创建的md文档里插入图片时，只需要将图片先复制到该同名文件夹中，然后只需要在你的博文中按照markdown的格式插入图片：
